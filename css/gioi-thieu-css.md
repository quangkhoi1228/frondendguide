# Giới thiệu CSS

## CSS là gì?

* CSS là viết tắt của Cascading Style Sheets
* CSS mổ tả cách phần tử HTML thể hiện trên giao diện như thế nào?
* CSS giúp chúng ta tiết kiệm thời gian vì có thể tái sử dụng lại CSS cho nhiều trang khác nhau.
